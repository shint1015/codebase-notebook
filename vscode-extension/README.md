# Codebase Notebook — VS Code extension

Ask your local [Codebase Notebook](../README.md) app about the current
selection or the whole workspace, straight from VS Code. Answers are grounded
in your indexed sources with citations, generated by your **local** model —
the editor bridge can never trigger an external API call.

## Requirements

- The Codebase Notebook desktop app must be running (it serves a
  token-protected API on `127.0.0.1:43110`).
- At least one workspace with indexed sources.

## Commands

- **Codebase Notebook: Ask about selection** — also in the editor context
  menu when text is selected. The selection is attached to your question.
- **Codebase Notebook: Ask workspace** — free-form question.

Answers open as a markdown preview beside your editor, with the cited
sources listed at the bottom.

## Build & install

```bash
cd vscode-extension
npm install
npm run compile
npx @vscode/vsce package   # produces codebase-notebook-vscode-x.y.z.vsix
code --install-extension codebase-notebook-vscode-*.vsix
```
